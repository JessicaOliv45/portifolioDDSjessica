Janela MDI

O projeto de janela MDI (Multiple Document Interface) consiste em uma aplicação que possui uma janela principal que atua como um contêiner para exibir múltiplas janelas filhas. Essas janelas filhas podem ser usadas para exibir documentos, ferramentas ou qualquer outro tipo de conteúdo que seja relevante para a aplicação.
